# Firebase Services

Firebase Auth, Storage ve client config **Adım 3'te** eklenecek.

Bu adımda şunlar yapılacak:
- `firebaseConfig.ts` — Firebase app başlatma
- `authService.ts` — Google/Email login, logout, token alma
- `storageService.ts` — Fotoğraf yükleme/silme işlemleri

Firestore direkt okuma yapılmayacak.
Tüm ilan verisi FastAPI üzerinden gelecek.
